Project Overview

This portfolio simulation demonstrates my ability to manage contractor procurement through a structured tender evaluation process. The project focuses on procurement strategy, contractor comparison, bid evaluation, and final contractor recommendation based on best overall value.

Procurement Tender Evaluation Commercial Awareness Contractor Selection Decision Making

Project Type

Residential Refurbishment Project

Role

Construction Project Manager (Portfolio Project)

Procurement Route

Traditional Design-Bid-Build

Tender Type

Competitive Tender

Status

Completed (Portfolio Simulation)

Focus

Procurement • Evaluation • Selection • Recommendation

Project Objectives

  • Define a suitable procurement strategy for internal fit-out works
  • Compare tender submissions using structured criteria
  • Evaluate contractors based on price, programme, experience, and quality
  • Select the contractor offering best overall value
  • Prepare a formal recommendation for appointment

Scope of Works

  • Preparation of procurement strategy
  • Issue of tender documentation to selected contractors
  • Comparison of submitted bids
  • Weighted evaluation of contractor proposals
  • Formal contractor recommendation for appointment

Key Deliverables

Procurement Strategy

A procurement strategy was prepared to define the tendering approach, procurement route, evaluation criteria, and selection method for the contractor appointment process.

  • Defined procurement route and tender method
  • Outlined tender process and contractor evaluation criteria
  • Established best-value contractor selection approach
Procurement Strategy (PDF)

Skills demonstrated: procurement planning, tender strategy, contractor selection method

Tender Comparison Table

Contractor tenders were compared against key commercial and delivery criteria, including submitted price, programme duration, relevant experience, and quality score.

  • Structured contractor comparison
  • Weighted scoring approach applied
  • Best-value contractor identified from tender returns
Tender Comparison Table (PDF)

Skills demonstrated: tender assessment, weighted scoring, commercial comparison

Bid Evaluation Report

A bid evaluation report was prepared to assess each tender submission and justify the contractor selection based on best overall value rather than lowest cost only.

  • Comparison of tender submissions
  • Evaluation of quality, price, programme, and experience
  • Formal recommendation supported by scoring results
Bid Evaluation Report (PDF)

Skills demonstrated: evaluation reporting, procurement analysis, contractor assessment

Contractor Recommendation

A formal recommendation was issued to support contractor appointment, confirming the selected bidder and summarising the reasons for selection.

  • Recommended contractor identified
  • Justification based on weighted scoring outcome
  • Formal procurement decision recorded
Contractor Recommendation (PDF)

Skills demonstrated: recommendation writing, contractor appointment, procurement decision-making

Project Outcomes

  • Procurement route and tender process clearly defined
  • Three contractor submissions evaluated through a weighted scoring system
  • Best-value contractor identified through structured comparison
  • Formal recommendation prepared to support contractor appointment

Lessons Learned

  • Lowest price does not always represent best project value
  • Weighted scoring improves transparency in contractor selection
  • Programme duration and quality should be assessed alongside cost
  • Formal evaluation reports strengthen procurement decisions

Tools & Methods Used

  • Excel / Google Sheets (Tender Comparison Table)
  • Word (Procurement Strategy, Bid Evaluation, Contractor Recommendation)
  • Weighted scoring method for contractor evaluation
  • Structured procurement documentation process

Why This Project Matters

This project demonstrates practical procurement and contractor selection skills. It shows how I assess tenders, compare suppliers, balance cost against quality and programme, and make structured recommendations that support project delivery.
